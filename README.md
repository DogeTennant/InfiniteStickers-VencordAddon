# InfiniteStickers - Vencord Plugin

A [Vencord](https://github.com/Vendicated/Vencord) plugin that gives you an unlimited personal sticker collection, hosted on GitHub and sent as real Discord sticker messages.

## The Problem

Discord limits custom stickers to 5 per server (or up to 60 with boosts), and Nitro's cross-server sticker feature still requires you to be a member of the server the sticker belongs to. With a large sticker collection this quickly becomes unmanageable - you either burn through your server slots or give up.

## How It Works

Stickers are stored as GIF/PNG files in a GitHub repository. When you send one:

1. The plugin fetches the GIF from GitHub
2. Uploads it temporarily to a small "transit" server you own
3. Sends it as a **real Discord sticker message**
4. Automatically removes it from the transit server once the slot is needed for a new one (rolling queue)

## Features

- **Unlimited stickers** hosted on GitHub, organized into categories by folder
- **Search** across your entire collection by name or tags - works across all categories simultaneously
- **Frequently Used** category that auto-populates and sorts by how often you use each sticker
- **Compose bar** - click a sticker, optionally type a message, send both together in one message
- **@mention support** - stage a sticker to the real Discord chat input for full autocomplete
- **Reply support** - pre-select a reply before opening the picker and it carries through
- **Auto-import** - scraper script downloads stickers from every server you're in
- **Live refresh** - pull new stickers from GitHub without restarting Discord

## Setup

### 1. GitHub Repository

Create a public GitHub repo with your stickers organized into subfolders:

```
your-repo/
  stickers/
    linkin-park/
      chester-laugh.gif
      mike-smile.gif
    reactions/
      facepalm.gif
  stickers.json         ← generated by the script
  generate-manifest.js  ← included in this repo
  scrape-stickers.js    ← included in this repo
```

Folder names become category names in the picker (hyphens/underscores are converted to spaces and title-cased automatically).

### 2. Generate the Manifest

```bash
node generate-manifest.js https://raw.githubusercontent.com/YOUR_USER/YOUR_REPO/main
```

Commit and push `stickers.json` to GitHub.

### 3. Transit Server

Create a small Discord server that you own. This is where stickers are briefly uploaded before being sent.

### 4. Install the Plugin

1. Clone [Vencord](https://github.com/Vendicated/Vencord) and run `pnpm install --frozen-lockfile`
2. Copy `index.tsx` into `src/plugins/infiniteStickers/index.tsx`
3. Run `pnpm build` then `pnpm inject`
4. Restart Discord

### 5. Configure

Go to **Settings → Plugins → InfiniteStickers → ⚙** and fill in:

| Setting | Description |
|---|---|
| Manifest URL | `https://raw.githubusercontent.com/YOU/REPO/main/stickers.json` |
| Transit Guild ID | Right-click your transit server → Copy Server ID (requires Developer Mode) |
| Transit Slots | Number of sticker slots in your transit server (default: 5) |

## Scraping Stickers from Discord

If your stickers are spread across Discord servers rather than on your computer, the included scraper can download them all automatically:

```bash
node scrape-stickers.js YOUR_DISCORD_TOKEN
```

> **Keep your token private** - it gives full access to your Discord account.

How to get your token: open Discord in a browser → F12 → Network tab → send any message → find a request to `discord.com/api` → look for the `Authorization` header.

After scraping, delete any folders you don't want, then regenerate the manifest and push.

## Adding New Stickers

1. Add GIF files to the appropriate folder in your repo (create new subfolders for new categories)
2. Run `node generate-manifest.js https://raw.githubusercontent.com/YOUR_USER/YOUR_REPO/main`
3. `git add . && git commit -m "Add stickers" && git push`
4. Hit the **↻** button in the picker - no Discord restart needed

## Notes

- Lottie-format stickers (Discord's animated vector stickers) are not supported and are skipped automatically
- This plugin is for personal use and works alongside Vencord's existing features
